Prework Study Guide

HTML

  • The head element contains information about the webpage. This information is typically known as metadata.
  • Meta elements contain information about the page that is used by the browser.
  • The body element represents the visible content shown to the user.
  • Elements nested inside other elements are known as child elements.
  • Element and Tag are typically used interchangeably but they are different.
  • Element refers to the entire element from opening tag to closing tag.
  • Tags refer to only what is inside the angled brackets.
  • 'a' element is known as an anchor element and it creates links to other web pages or the same web page.

CSS

  • CSS stands for Cascading Style Sheet.
  • CSS provides the visual layout of the web app.
  • It is used to style the HTML elements.

Git

  • Git is a version control system that stores the different versions of your program.
  • Git commands in the Command Line Interface always start with "git".
  • git status: checks what branch we are currently on.
  • git checkout -b "branch-name": creates a new branch and switches to it.
  • Git Hub is the cloud-based hosting service that allows us to keep track of our projects inside repositories (aka repos).
  • Repositories (Repos) are used as folders to store files for your project.
  • Git Hub is also used to collaborate with other developers.
  • Branches are created to work on the project without modifying the live version.
  • When you are done modifying the project you can submit a Pull Request to merge the branch you were working on into the main branch.
